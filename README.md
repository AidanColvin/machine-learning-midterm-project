# Kaggle Competition Project

> **Submission Deadline:** March 8 at 11:59:59 PM EST — No extensions.

Submit a **3-page PDF** summarizing your Kaggle competition project.

---

## Rubric

### 1. Problem Description — 20 points

| Criterion | Points | Description |
|---|---|---|
| Clarity of problem statement | 5 | Clearly articulates the problem being solved by the competition. |
| Dataset description | 5 | Detailed description including sample size, number of features, and the response variable. The meaning of each feature/predictor is clearly explained. |
| Objective | 5 | Clearly states the competition objective and what success looks like (e.g., the metric(s) used to evaluate submissions). |
| Relevance to field | 5 | Briefly discusses the relevance and potential impact of solving the problem in the context of health or other scientific fields. |

---

### 2. Methodology — 30 points

| Criterion | Points | Description |
|---|---|---|
| Overview of approaches | 10 | Describes all methods attempted, including those covered in class and those learned independently. |
| Rationale for chosen method | 10 | Explains why the final method was chosen over others, including suitability for the problem, dataset, and any constraints (e.g., computational resources). |
| Implementation details | 5 | Provides sufficient detail on implementation (e.g., data preprocessing, model selection, hyperparameter tuning) to understand how the project was carried out. |
| Reproducibility | 5 | Includes details (e.g., code snippets, software versions) that would allow someone else to reproduce the work. |

---

### 3. Results and Evaluation — 25 points

| Criterion | Points | Description |
|---|---|---|
| Performance metrics | 10 | Clearly presents results using appropriate metrics. Includes a discussion on the choice of metrics. |
| Analysis of results | 10 | Thorough analysis including strengths and weaknesses of the model(s). Discusses interesting findings or unexpected outcomes. |
| Comparative analysis | 5 | If applicable, compares the chosen method against other methods or baselines using clear figures, tables, and numbers. |

---

### 4. Presentation and Writing Quality — 15 points

| Criterion | Points | Description |
|---|---|---|
| Organization and structure | 5 | Well-organized with a logical flow that is easy to follow. |
| Clarity of writing | 5 | Clear, concise, and free of grammatical or spelling errors. Technical terms are adequately explained. |
| Visuals and tables | 5 | Effectively uses figures, tables, and charts to illustrate findings. Visuals are clear, well-labeled, and appropriately referenced in the text. |

---

### 5. Creativity and Innovation — 10 points

| Criterion | Points | Description |
|---|---|---|
| Novelty of approach | 5 | Demonstrates creativity or innovation (e.g., new methods, creative preprocessing, or novel application of a known method). |
| Insightfulness | 5 | Shows depth of insight in analyzing the problem and interpreting results. Offers thoughtful considerations on implications for the field of biostatistics. |

---

## Total: 100 points
