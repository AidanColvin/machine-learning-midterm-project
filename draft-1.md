# Part 1 - Problem Description

Given 13 features from an individual, predict the probability that they have heart disease. The dataset was synthetically generated by a deep learning model trained on a real heart disease prediction dataset. Feature distributions closely mirror real-world observed data but are not exact replicas. The training dataset had a sample size of n=630,000 across 13 features with no missing values.
The response variable is Heart Disease, labeled 0 (Absence) or 1 (Presence) in the training data. Once the model is applied to the test data, the output is the predicted probability that an individual has heart disease. The 13 features fall into three types: continuous, binary, and ordinal. 

| Feature | Type | Description |
|---------|------|-------------|
| Age | Continuous | Patient age in years |
| Blood Pressure | Continuous | Resting blood pressure (mmHg) |
| Cholesterol | Continuous | Serum cholesterol (mg/dl) |
| Max Heart Rate | Continuous | Maximum heart rate during stress test |
| ST Depression | Continuous | EKG reading measuring reduced blood flow during exercise |
| Sex | Binary | 0 = Female, 1 = Male |
| Fasting Blood Sugar | Binary | Blood sugar over 120 mg/dl — 0 = No, 1 = Yes |
| Exercise-Induced Angina | Binary | Chest pain during exercise — 0 = No, 1 = Yes |
| Chest Pain Type | Ordinal | 1=Typical Angina, 2=Atypical Angina, 3=Non-Anginal, 4=Asymptomatic |
| EKG Results | Ordinal | 0=Normal, 1=ST-T Abnormality, 2=Left Ventricular Hypertrophy |
| Slope of ST | Ordinal | 1=Upsloping, 2=Flat, 3=Downsloping |
| Number of Vessels Fluoroscopy | Ordinal | Major coronary vessels visible under dye (0–3) |
| Thallium Stress Test | Ordinal | 3=Normal, 6=Fixed Defect, 7=Reversible Defect |

The goal is to predict the probability that an individual has heart disease, scored using AUC. A score of 1.0 is perfect; 0.5 is random. Heart disease is the leading cause of death worldwide. Identifying high-risk individuals early using routine clinical data has direct potential to improve patient outcomes.
